# github-chatmodes
GitHub Copilot のカスタムチャットモード用の Markdown ファイルを格納したレポジトリです

#### azure-policy.chatmode.md
Azure Policy を 'Policy as Code' の観点で作成、レビュー、テスト、改善するためのカスタムチャットモード。ポリシー定義(JSON/Bicep/ARM/Terraform) の品質チェック、スコープ設計、影響評価、サンプル実装、AZ CLI/ARM/Bicepでの検証手順を提供します

詳細は以下の記事を参照してください
- [Custom chat modes in VS Code](https://code.visualstudio.com/docs/copilot/customization/custom-chat-modes#_create-a-chat-mode)
- [ベストプラクティスのドキュメントからGitHub Copilot のカスタムチャットモードを生成してみる](https://zenn.dev/yurio/articles/7b2082a0ab2ea8)
